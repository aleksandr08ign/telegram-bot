Spring Boot приложение для создания и автоматической отправки напоминаний через Telegram бота.

Как пользоваться:
Найдите бота в Telegram
Отправьте сообщение в формате:
text
01.01.2025 20:00 Сделать домашнюю работу
Бот пришлет напоминание в указанное время

Команды:
/start - приветствие и инструкция
/my_tasks - список ваших напоминаний

Технологии:
Java 21 + Spring Boot 2.7.9
Spring Data JPA + H2 Database
Telegram Bot API
Liquibase (миграции БД)

Конфигурация:
Токен бота: через переменную окружения BOT_TOKEN
БД для разработки: H2 (http://localhost:8080/h2-console)
готова конфигурация для PostgreSQL

Безопасность:
Токены не хранятся в коде
Конфигурация через переменные окружения
